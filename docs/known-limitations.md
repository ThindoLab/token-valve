# Known Limitations And Backlog

## MVP Limitations

- macOS-first.
- Source install only; no npm package or Homebrew tap yet.
- No full Web UI.
- No team cloud vault.
- No hostile-agent sandbox.
- No Linux Secret Service or Windows Credential Manager implementation yet.
- No 1Password or Bitwarden backend yet.
- No GUI confirmation prompt.
- MCP and Skill wrappers may need host-specific packaging.
- Custom provider support is intentionally data-driven but still early.

## Backlog

- Local Web UI for “open the key manager”.
- Linux Secret Service.
- Windows Credential Manager.
- 1Password CLI backend.
- Bitwarden CLI backend.
- Encrypted SQLite backend with a clear root-key story.
- GUI or native confirmation prompts.
- Team policy templates.
- More provider adapters, such as Stripe and Cloudflare.
- Stronger containment for hostile-agent scenarios.
- Published npm package.
- Homebrew tap.
